## Revision Tool
In my 3rd year of University, for my disseration, I chose to develop a program that generates revision questions using the users notes in order to revise. This utilised the openAI api to implement a large language model to use prompt engineering in order to generate questions. This LLM also utilised different revision techniques such as adaptive weights in order to test the user on their weaker topics. This is achieved by recording all question responses and their accuracy in each topic, then using this to identify weaker topics and weigh them to be more likely in question generation.
